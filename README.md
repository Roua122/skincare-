# Botanical Glow: Premium Skincare E-Commerce App 🌿

Welcome to **Botanical Glow**, a modern, cross-platform E-Commerce mobile application designed for skincare enthusiasts. This project was built using **Flutter**, leveraging **Firebase** for backend services and **Provider** for efficient state management.

---

## 🚀 Key Features

* **Authentication:** Secure User Sign-up & Sign-in with Firebase Auth.
* **Real-time Database:** Sync your favorites and shopping cart across all devices instantly using **Cloud Firestore**.
* **State Management:** Built with **Provider** for clean, scalable, and reactive data handling.
* **Modern UI/UX:** * Glassmorphism design elements.
    * Smooth animations and transitions.
    * Cached network images for high-performance loading.
    * Offline mode handling for seamless user experience.
* **Product Management:** Dynamic categories, detailed product view, and responsive product grid.

---

## 🛠 Tech Stack

* **Framework:** Flutter (Dart)
* **Backend:** Firebase (Authentication, Cloud Firestore)
* **State Management:** Provider
* **Image Handling:** `cached_network_image`
* **Architecture:** Clean modular architecture (presentation, data, providers, models)

---

## 🏗 Getting Started

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/Roua122/flutter-skincare-app.git](https://github.com/Roua122/flutter-skincare-app.git)

```

2. **Navigate to the project directory:**
```bash
cd flutter-skincare-app

```


3. **Install dependencies:**
```bash
flutter pub get

```


4. **Run the application:**
```bash
flutter run

```

## 💡 Key Challenges & Solutions

* **State Synchronization:** Used Firebase `snapshots()` listeners to ensure real-time UI updates across different devices.
* **Performance:** Implemented cached image rendering to minimize network usage and improve scroll speed.
* **Error Handling:** Custom authentication error handling to provide a user-friendly experience in English.
